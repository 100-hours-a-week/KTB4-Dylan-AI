# sweep

파일을 확장자별 폴더로 정리하는 작은 CLI입니다.

## 과제 아이디어

다운로드 폴더처럼 여러 파일이 섞이기 쉬운 폴더를 확장자 기준으로 자동 정리하는 도구를 만들었습니다.
실수로 파일이 바로 이동되지 않도록 기본 실행은 미리보기만 하고, `--apply` 옵션을 붙였을 때만 실제로 이동합니다.

## 실행 방법

```bash
uv run python main.py
```

특정 폴더를 보려면:

```bash
uv run python main.py ~/Downloads
```

실제로 이동하려면:

```bash
uv run python main.py ~/Downloads --apply
```

## 회고

단순히 파일을 옮기는 기능뿐 아니라, 사용자가 실수하지 않도록 미리보기 흐름을 넣는 것이 중요하다고 느꼈습니다.
또한 같은 이름의 파일이 있을 때 건너뛰는 처리처럼 작은 프로그램에서도 예외 상황을 고려해야 한다는 점을 배웠습니다.
